# App de Domicilios en C

Aplicación de consola desarrollada en lenguaje C que simula un sistema básico de pedidos de comida.

## 📋 Funcionalidades

* Ver menú de productos
* Agregar productos al carrito
* Ver carrito y total
* Confirmar pedido
* Salir del programa

## 🛠️ Tecnologías

* Lenguaje C
* Uso de estructuras (`struct`)
* Condicionales (`if`, `switch`)
* Ciclos (`while`)

## ▶️ Cómo ejecutar

1. Compilar el programa:

```
gcc main.c -o app
```

2. Ejecutar:

```
./app
```
